# Line_AI_Bot

- line apiとapi gatewayを連携させる  
- できること  
  - 英語が記述されたテキストの画像をラインボットアカウントに送信  
  - すると画像からテキストを抽出しさらに翻訳して結果をリターンする  
- 技術  
  - AWS  
    - Lambda  
    - API Gateway
    - Amazon Translate  
    - Amazon Transcribe  
    - Cloudformation
   - 言語  
     - javascript
